# Tower Plan

Status: scaffolded

## Objective

Collect a bounded set of failed DAO, crypto, token, bounty, governance, treasury, and exploit precedents that can stress-test the ResonantDAO economy design.

## Closure Standard

- Each precedent names the mechanism that failed.
- Each precedent maps to a whitepaper pressure point.
- Anecdotes stay distinct from evidence-backed patterns.
- Famous failures that do not apply are rejected rather than forced.
