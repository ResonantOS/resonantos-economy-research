# L1 Residue Map

Status: scaffolded

| Formal residue | Question |
| --- | --- |
| Conservation | Does credit allocation sum to a bounded budget? |
| Anti-collusion | Can reciprocal review, team farming, and circular contribution be detected? |
| Randomness | Is unpredictability used for audit defense, not arbitrary payout? |
| Signal separation | Does a score route attention without deciding authority? |
| Temporal decay | Does reputation reflect recent contribution without erasing durable work? |
| Stress threshold | What breaks under liquidity, treasury, or holder-concentration shocks? |
