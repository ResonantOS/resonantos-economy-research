# Socioeconomic Behavior Subtower

Status: scaffolded
Promotion scope: local-research-only

## Boundary

This subtower studies socioeconomic mentalities, behavioral economics, motivation, trust, identity, status, cooperation, community building, and how reward systems train behavior over time.

It should explain how mechanisms shape people, not only how people respond to incentives.

## Research Questions

- What participant mentality does each reward mechanism create?
- Which incentives crowd out intrinsic motivation or community identity?
- What rituals, review processes, and stewardship practices keep contribution from collapsing into task extraction?
