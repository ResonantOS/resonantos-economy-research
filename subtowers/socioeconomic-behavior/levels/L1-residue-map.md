# L1 Residue Map

Status: scaffolded

| Behavioral residue | Economy warning |
| --- | --- |
| Motivation crowding | Paying for everything can reduce unpaid stewardship. |
| Status competition | Reputation can become a status game even if non-transferable. |
| Fairness perception | Random or opaque rewards may feel illegitimate. |
| Identity split | Contributors may become mercenaries, members, stewards, or spectators depending on rituals and incentives. |
| Review fatigue | Heavy governance can burn out the very contributors it needs. |
| Trust decay | Fraud controls can become surveillance if not bounded. |
