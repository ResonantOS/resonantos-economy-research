# Official Whitepaper Subtower

Status: scaffolded
Promotion scope: local-research-only

## Boundary

This subtower has one primary source: `https://resonantdao.com/whitepaper/`.

It records what the whitepaper says and what local research questions follow from it. It must not import external DAO criticism as if it were part of the whitepaper.

## Research Questions

- What economy mechanisms does the whitepaper actually propose?
- Which mechanisms are Phase 1, Phase 2, or future research?
- Which official claims need outside evidence before implementation?
- Which claims already contain their own gates or caveats?

## Mechanism-Directed Prior Art

For a proposed mechanism, use the shared
[mechanism-directed prior-art contract](../MECHANISM-PRIOR-ART-CONTRACT.md) and
ask: **what does the official source actually support, leave open, or prohibit
about this mechanism?** This lane admits only the official whitepaper. Its
required negative is an unsupported inference, missing rule, internal caveat,
or phase boundary—not an external criticism imported as official intent.
