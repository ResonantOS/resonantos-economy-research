## Summary

The repository has a local agent-name pool for future governed dispatches. The
pool is tuned to the current research lanes: official source reading, failed
DAO/crypto precedent, science of science, formal mechanisms, behavioral
economics, synthesis, skepticism, writing, audit, and workflow reflection.

## Residue

The pool is small and should be expanded only when a dispatch or reflection
loop needs a new role fit.

## Promotion Boundary

Agent names are dispatch aids, not authority. No agent can self-approve work it
helped produce.
