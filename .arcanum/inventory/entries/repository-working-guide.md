## Summary

The repository working guide defines the local posture for work in this tower:
source-kind separation, signal-credit-authority separation, witness-first
mechanism evaluation, visible residue, private/internal consultation posture,
and the continuation route after the first harvests and fixture scaffold.

## Reusable Selectors

- `repository-posture` for contributor and agent orientation.
- `continuation-surface` for resuming research from harvested selectors,
  subtower returns, and contribution-edge fixtures.
- `dispatch-readiness` for deciding whether a new or resumed fan-out is ready
  to register and run.
- `private-internal-public-export-gate` for keeping local provenance and
  telemetry out of public derivatives until scrubbed.

## Residue

The guide is current for internal continuation, but public release still
requires a dedicated scrub and derivative package.

## Promotion Boundary

The guide is an operating posture. It does not authorize economy mechanisms,
token policy, bounty policy, governance authority, or implementation readiness.
