## Corrected Direct Result

The planned `1,000,000,000 RES` whitepaper value is **not a constraint**. The
operator has decided that it will be replaced, so selecting the initial total
supply is inside the Number Lottery project.

The strongest current candidate is not a raw character sum. It is a transparent
collective draw:

1. freeze the supply band and all participation/mapping rules;
2. admit public artifacts or commitments to non-public capsule artifacts;
3. freeze and publish the canonical cohort root;
4. mix that root with a predeclared post-close public beacon;
5. map the domain-separated seed into the frozen band;
6. publish the resulting initial total supply and reproduction receipt.

This lets the number be arbitrary by design and meaningful through community
provenance, while making post-result rule changes visible.
