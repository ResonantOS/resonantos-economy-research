# Local Fixture Runner

Serve the `site/` directory locally, then open:

`http://127.0.0.1:8127/contribute/tests/fixture-runner.html`

The runner loads the existing simulator in a same-origin hidden frame and
exercises fixture profile `resonant-time-capsule-simulator@1.0.0`. It uses only
invented values. A machine `PASS` means the current browser client matches the
declared local grammar, states, and events; it does not complete the manual
access and consequence-retell matrix in `ACCESS-MATRIX.md`.

No fixture or result is a production intake, cryptographic, privacy, security,
durability, fairness, token, deployment, or launch artifact.
