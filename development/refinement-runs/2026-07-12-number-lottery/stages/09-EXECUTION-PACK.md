# RCBC Execution-Pack Handoff

Status: metadata-only; execution not authorized

## Choreography

```text
W0 specification contracts
  -> human/owner review
  -> W1 deterministic public toy
  -> evidence adjudication
  -> W2 stewardship/event model
  -> owner review
  -> W3 commitment criterion
  -> separate security evidence
  -> W4 readiness audit
```

Parallelism is intentionally limited. Inside W0, owner/interface collection may
run beside source-snapshot drafting only after both use the same frozen glossary.
Fixture encoding cannot run before the common envelope. No later wave begins
from a narrative “looks good” verdict; it requires prior-layer evidence.

The preferred runtime handoff is one SWU at a time through Context Builder and
Task Session. Subagent execution is a recommendation per SWU, not pre-approved
by this plan.
