# Video Concept Map

## Governing Question

How can one personal message become part of a number shared by everyone—and still return later?

## Concept Sequence

| Order | Concept | Working meaning | Visual witness | Not the same as |
| ---: | --- | --- | --- | --- |
| 1 | Something for later | A message or small mark directed toward a future opening. | One folded parcel held close. | A vote, payment, or token purchase. |
| 2 | Open space | Participation remains voluntary; nothing has to be submitted. | One quiet point does not launch a parcel. | An empty record invented by the system. |
| 3 | Shared capsule | Accepted records join one collective container. | Many paths converge into one circle. | One number per person. |
| 4 | Shared fingerprint | The closed capsule becomes one digital trace of the whole. | Concentric geometry forms around the capsule. | Visible message text or personal score. |
| 5 | Public closing signal | One later public value joins the fingerprint at closing. | A single external pulse reaches the capsule. | Message storage. |
| 6 | Starting token number | The shared result helps choose how many tokens ResonantDAO starts with. | The capsule’s light resolves behind editorial digits. | Allocation to contributors. |
| 7 | Future return | The number is chosen once; messages continue to 2, 5, and 10 years. | Three paths leave the fixed center. | Three new number selections. |

## Honesty Boundary

`42` is a narrative easter egg. The voice does not name it. While the digits remain
visible, the voice says the real number comes later from the capsule shaped together.
