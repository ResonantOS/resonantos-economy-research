# Glossary Consistency

| Term | Meaning in this design | Rejected drift |
| --- | --- | --- |
| short message | one brief phrase or sentence; one accepted record | a message weighted by length |
| opening | the 2, 5, or 10-year community return | a new number selection |
| share from the beginning | message may be public from capsule publication | instant unmoderated publication |
| saved for this opening | message text withheld; a commitment enters the manifest | guaranteed privacy or recovery |
| commitment | fingerprint made from message plus a random secret | encryption or message storage |
| capsule fingerprint | fingerprint of the frozen master manifest | one person's score |
| Bitcoin closing block | block containing/after the anchor transaction under frozen rules | storage for message text |
| ritual number | symbolic community result derived once | personal allocation or economic validation |

Result: consistent within this design. Earlier word-based artifacts need the patch requests
recorded in `UPSTREAM-PATCH-REQUESTS.md` before reuse.
