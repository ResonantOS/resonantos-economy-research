# Distill Design-Unit Validation

Mode: Standard Distill

Status: `pass` for the candidate design unit; `flag` for human proof

## Selected Unit

**Narrative SCU**

> Leave words. Shape one beginning. Meet them again.

**Experience SCU**

One compact story followed by one stable form shell with three progressive
panels: Words → Choices → Review.

## Coherence

The unit answers the complete minimum story:

- action: optional words;
- collective effect: one capsule-wide pattern helps select one starting
  supply;
- time: number once, words at 2/5/10 years;
- agency: public choice where supported and quiet without penalty;
- truth: browser-only rehearsal.

Removing any of these changes the concept. Adding credentials, moderation,
hash construction, crypto custody, database behavior, or token execution would
expand the first-pass unit beyond the comprehension target.

## Recomposition Proof

```text
narrative SCU
  -> hero sentence + timing correction
  -> visual equation
  -> Words panel
  -> Choices panel
  -> Review panel
  -> mechanism disclosure
  -> rehearsal receipt
```

Each surface carries the same model without changing timing, reward,
optionality, or privacy claims. The unit can later connect to admission,
intake, capsule-root, supply-selection, and release components by handle; it
does not absorb their authority.

## Split Pressure

| Pressure | Kept outside the unit | Interface retained |
| --- | --- | --- |
| unique participation | credential issuance and personhood | future admission boundary |
| live collection | API, database, edits, close transaction | future state/receipt boundary |
| saved-for-later content | encryption, key custody, recovery, migration | disabled explanatory choice |
| supply execution | token contract and mint authority | “candidate starting number” only |
| moderation | policy, appeals, display disposition | public-content consequence only |

## Premortem

Most likely failure: people remember the poetic time-capsule idea but cannot
explain how it affects the number—or they infer a personal token reward.

Guardrail:

- keep the token consequence in the hero lead;
- keep “number once / words later” as a standalone line;
- place the no-personal-reward correction before the form;
- require unprompted retell evidence before publication.

## Frame Expiry

Rerun the design if any of these change:

- 2/5/10-year horizons;
- two-word maximum;
- initial supply effect;
- one-time selection timing;
- quiet/no-penalty contract;
- public or saved-for-later meaning;
- rehearsal collection boundary;
- audience or primary destination.

## Verdict

The unit is small enough to understand and complete enough to recompose. The
authoring verdict is `pass`; the experience claim remains `flag` until the
five-person retell and accessibility gates run.
