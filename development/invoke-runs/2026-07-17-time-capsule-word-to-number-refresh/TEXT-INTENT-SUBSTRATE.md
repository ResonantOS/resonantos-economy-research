# Whisper Text Intent Substrate — Mechanism Clarity

Status: `pass`

Transport: `interactive-web-copy-extension` (`candidate` / unproven)

## Resonance Core

The mechanism should feel transparent rather than magical. A small human word
enters a shared, publicly checkable process without becoming a personal token
claim.

Tone: direct, curious, calm, concrete.

Desired residue:

> I can see exactly where the word enters and why nobody knows the final number
> yet.

Forbidden feels:

- a black box dressed in poetic language;
- a letter puzzle or numerology system;
- gambling, winning, or personal allocation;
- technical intimidation;
- false certainty about an unvalidated production mechanism.

## Relevance Core

Target public: ResonantDAO community and founding contributors who already hear
that words influence the number but cannot explain the causal steps.

Reader question:

> But how can a word become a token number?

Required correction:

> It does not become a number by itself. It becomes a record that changes the
> fingerprint of the whole capsule.

Objections to answer:

- Are letters assigned numbers?
- Are word values added together?
- Is my word worth tokens?
- Can the organizers choose the result after seeing the words?
- Why is another public value needed?
- What does the pre-agreed range do?

## Trajectory Core

```text
ask the exact question
  -> reject direct word value
  -> show word becoming record
  -> show all records becoming capsule fingerprint
  -> pause because later public value is missing
  -> add invented later value
  -> reveal toy result inside frozen range
  -> compare one different word
  -> return to time-capsule meaning and participation
```

## Smallest Coherent Text Unit

> Every word becomes one capsule record. Every record changes the capsule's
> fingerprint. After closure, that fingerprint and a new public value select
> one number inside the agreed range.

## Proof Boundary

- “Fingerprint” is a plain alias for the candidate capsule root.
- The later public source, range, and mapping must be chosen and frozen before
  real intake but are not selected here.
- The implementation example must compute its values from published invented
  fixtures; the copy must not contain decorative fake outputs.
- Human comprehension remains pending.
