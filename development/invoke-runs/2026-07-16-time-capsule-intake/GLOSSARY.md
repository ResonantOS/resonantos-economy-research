# Resonant Time Capsule Working Glossary

Status: Invoke-local candidate vocabulary; no canonical promotion

| Term | Meaning in this run | Link status |
| --- | --- | --- |
| Admitted participant | A participant for whom the external admission owner issued one usable credential. It does not mean proven unique human. | partial |
| Participation credential | Opaque quota-enforcement handle used by intake and omitted from the public manifest. | new candidate |
| Release horizon | One of the fixed 2-, 5-, or 10-year capsule openings. | linked to earlier `capsule epoch`; renamed for public clarity |
| Word slot | One of two ordered optional places belonging to one horizon in a participant envelope. | new candidate |
| Submission envelope | Private intake state grouping one credential's six optional slots. | new candidate |
| Public-now record | Capsule record whose normalized word may be displayed from the beginning. | partial |
| Sealed-until-release record | Capsule record containing approved cryptographic references, not plaintext, and eligible for later opening under the release consent policy. | partial; cryptographic profile open |
| Capsule manifest | Frozen canonical list of accepted records across all three horizons. | linked to earlier cohort manifest |
| Capsule root | Authenticated digest of the frozen manifest. It proves neither identity, consent, fairness, nor economics. | linked |
| Post-close beacon | Predeclared public value observed only after intake closure and used in the supply seed. | new candidate |
| Supply selection rules | Frozen versioned contract for band, records, closure, beacon, and mapping. | new composition |
| Initial supply receipt | Reproduction record containing rules digest, root, beacon, mapping version, and result. | new candidate |
| Release manifest | Horizon-specific view of public words, authorized openings, non-openings, and display dispositions. | new candidate |
| Display disposition | Separate release-time decision to show, quarantine, or withhold content without changing the frozen root. | partial |

No term here updates repository definitions, canonical glossary, or Inventory authority.
