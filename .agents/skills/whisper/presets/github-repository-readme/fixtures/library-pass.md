# Pebble

Pebble is a tiny example library that turns a list of words into a stable,
sorted list.

## Try It

Run:

```bash
pebble sort alpha beta
```

You should see `alpha` followed by `beta`. Stop if the command is unavailable;
the [fixture notes](README.md) are the only deeper route in this fictional case.

## Limit

This is a structural fixture. It is not a real package or evidence that a reader
can use one.
