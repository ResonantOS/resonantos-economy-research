# Human Editorial Pair — Situation Before Label

This pair is a human editorial fixture, not a machine comprehension test.

## Failing Candidate

> Forty fictional unfamiliar reviewers classify twelve counterbalanced invented
> event cases. Compare a public recap with the same recap plus bounded evidence.

The sentence leads with protocol design and makes the reader decode the setup
before knowing the ordinary problem.

## Passing Candidate

> Imagine an event where the talk went well, but nobody noticed the person who
> found the accessible room and fixed the schedule. Show one group only the
> public recap. Show another group the same recap plus a short note about the
> hidden work. Then ask both groups who helped and why.

Only after the situation is understood does the deeper procedure need labels
for cases, groups, or comparison rules.

Passing here means “matches the declared editorial ordering on inspection.” It
does not mean a newcomer understood it; that still needs a held-out reader.
